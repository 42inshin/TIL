# 메뉴 만들기

# 알아야할 css 항목

  • 메뉴 최소/ 최대 너비: min-width, max-width
  • 메뉴 너비 균등: display: table, table-layout
  • 보더 겹침 현상: margin: -1px

# 1단 메뉴 만들기

# HTML

<div class="wrap">
		<ul class="menu">
			<li class="menu_item"><a href="#" class="menu_link">메일</a></li>
			<li class="menu_item"><a href="#" class="menu_link">카페</a></li>
			<li class="menu_item"><a href="#" class="menu_link">소프트웨어</a></li>
			<li class="menu_item"><a href="#" class="menu_link">지식백과</a></li>
			<li class="menu_item"><a href="#" class="menu_link">영화</a></li>
		</ul>
	</div>

# CSS

.menu {
		display: table;
		table-layout: fixed;
		width: 100%;
	}
	.menu_item {
		display: table-cell;
	}
.menu_link {
		display: block;
		height: 36px;
		line-height: 36px;
		border: 1px solid #ddd;
		font-size: 12px;
		color: #333;
		text-align: center;
	}
	.menu_item + .menu_item .menu_link {
		margin-left: -1px;
	}
	.menu_item:hover .menu_link {
		color: green;
		font-weight: bold;
	}

# 한줄 말처리 CSS

.panel_item a {
		font-size: 14px;

		display: block;

# white-space: nowrap

: wrap 하지말고 텍스트를 한줄로 보이게 한다.

# text-overflow: ellipsis

: box에 넘어가는 경우 ... 으로 처리한다. overflow:hidden 필요!

# 7. 학습 내용에 대한 개인적인 총평

  • 기존에 매번 하던 html, css 라서 쉽게 따라할 것이라 생각했었는데, 강의를 살펴보면서 새롭게 알게되는 점들이 많다. 특히 두줄 말처리와 같이 생각보다 자주 쓸만한 것들을 작업해본적이 없어서 많은 도움이 되었다. 계속해서 프론트엔드 쪽 공부를 진행하여 빠르게 취업준비를 하도록 하자. 지금의 시간을 더 알뜰하게 사용해야 5월달에 취업이 가능할 것이다. 화이팅하자.
