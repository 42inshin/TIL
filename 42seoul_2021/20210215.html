# 크롤링 방법2

: 이 방법은 브라우저 자체를 움직여서 다운받는 형식으로 위와 같은 오류를 방지할 수 있다.

  1. 파이썬 가상환경 만들기
    python -m venv selenium(여기서 selenium은 만드는 폴더명이다.)
  2. 가상환경을 켜기위해서 bin 폴더로 이동
    cd selenium/bin
    윈도우는 Scripts 폴더로 이동
  3. 가상환경 실행
    mac os : source activate
    window : activate
    가상환경에 있는경우 터미널에 (selenium) 이라고 뜸
  4. 가상환경에서 selenium 설치 pip install selenium
  5. chromedrive 설치 chromedrive 검색 후 celenium 폴더에 chromedrive.zip 저장 후 압축풀기
  6. 관련 코드 붙여넣기(python) 링크 : https://sites.google.com/a/chromium.org/chromedriver/getting-started
  7. 스크롤 관련 코드 찾아 붙여넣은 후 맥락에 맞게 수정

# 크롤링 2번째 상세코드

import time
from selenium import webdriver

driver = webdriver.Chrome('/Users/ninos/Desktop/crawling/selenium/chromedriver')  # Optional argument, if not specified will search path.
driver.get('https://www.google.co.kr/imghp?hl=ko&tab=ri&ogbl')
# time.sleep(3) # Let the user actually see something!
search_box = driver.find_element_by_name('q')
search_box.send_keys('아이유')
search_box.submit()

SCROLL_PAUSE_TIME = 1
# Get scroll height
last_height = driver.execute_script("return document.body.scrollHeight")
while True:
    # Scroll down to bottom
    driver.execute_script("window.scrollTo(0, document.body.scrollHeight);")
    # Wait to load page
    time.sleep(SCROLL_PAUSE_TIME)
    # Calculate new scroll height and compare with last scroll height
    new_height = driver.execute_script("return document.body.scrollHeight")
    if new_height == last_height:
        try:
            driver.find_element_by_css_selector(".mye4qd").click()
        except:
            break
    last_height = new_height

images = driver.find_elements_by_css_selector(".rg_i.Q4LuWd")
count = 1
for img in images:
    img.click()
    time.sleep(2) # Let the user actually see something!
    imgUrl = driver.find_element_by_xpath("/html/body/div[2]/c-wiz/div[3]/div[2]/div[3]/div/div/div[3]/div[2]/c-wiz/div/div[1]/div[1]/div/div[2]/a/img").get_attribute("src")
    import urllib.request
    urllib.request.urlretrieve(imgUrl, str(count) + ".jpg")
    count = count + 1

# 7. 학습 내용에 대한 개인적인 총평

  • 구글링을 통하여 파이썬 코드를 복사하고, 그 코드를 재편집하여 이미지 크롤링을 진행해보았다. 첫번째 방법은 이미 만들어진 파이썬 google_images_download 를 설치하여 간단하게 이미지를 크롤링 하는 방법이었으며, 두번째는 직접 브라우저를 작동시켜 크롤링을 하는 방법이었다.
  • 첫번째 방법은 속도가 빠르나, 시간이 지남에 따라 프로그램이 작동하지 않을 가능성이 높았고, 두번째 방법은 상대적으로 속도가 느리지만 지속적으로 사용할 수 있는 코드였다.
  • 두번째는 직접 javaScript를 다루고, css 선택자를 이용하여 다운받는 방법이여서 속도면에 있어서 좋지는 않지만, 다양한 방법으로 변형이 가능하여 웹에서 동작하여 필요한 일들을 자동화 하는 것에 유용할 것으로 판단 된다.
