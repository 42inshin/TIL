# 햄버거 버튼

햄버거 버튼은 주로 모바일에서 사용하는 버튼이다. 이미지로 만들어도 되지만, 간단하게 코드로 작성하는 방법을 익혀두자.

.gnb_menu {
		position: absolute;
		top: 12px;
		left: 12px;
		display: block;
		width: 40px;
		height: 35px;
		font-size: 1px;  /* 글자 없애기 */
		color: transparent; /* 글자 없애기 */
		background: linear-gradient(#fff 50%, transparent 50%);
		background-size: 100% 10px;
	}
  • 가로 40px, 세로 35px 사각형을 만든후 linear-gradient 와 bg-size 를 이용하여 모양을 만들어주자.

# form 태그, placeholder

placeholder 에도 css 값을 변경할 수 있다. placeholder 의 CSS 값을 변경하기 위해선 prefix(접두사) ::placeholder 를 사용해야한다.

비표준 코드이기 때문에 브라우저 호환을 위해 예시처럼 작성한다.

  • 예시
::-webkit-input-placeholder { /* Edge */
  color: red;
}

:-ms-input-placeholder { /* Internet Explorer 10-11 */
  color: red;
}

::placeholder {
  color: red;
}

screen and (max-height:375px) and (orientation: landscape)

# Select 커스텀

아래로 화살표를 없애려면, appearance: none; 을 써줘야 한다.

appearance 값은 비표준 명령어 이기 때문에 -wekit- 등과 같은 prefix를 써줘야 한다.

.select_form select {
		-webkit-appearance: none;
		-moz-appearance: none;
	}

# 7. 학습 내용에 대한 개인적인 총평

  • 전날 학습을 밤늦게까지 진행하다보니 몸 컨디션이 망가졌다. 몰입을 하는 것을 괜찮은데 건강을 생각해서 적절하게 학습을 진행하도록 컨디션 조절을 잘 해야겠다.
  • html, css 를 간만에 잡다보니, 쉽게 처리하던 부분들에서 시간을 많이 쓰게 되었다. 계속해서 사용을 안하다보니 확실히 속도와 처리방법에 대해서 느려지는 경향이 보인다. 이번 기회를 통해서 꾸준히 습득하여 내꺼로 만들도록 하자.
