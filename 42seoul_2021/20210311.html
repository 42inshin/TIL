# 함수

  • 함수에 이름이 필요없을 경우 안적어도 사용 가능. (객체로 함수 불러오자.)

# 전역변수, 지역변수

  • 전역변수 : <script>안 최상위에 변수를 선언한 것. 다른 함수에서 사용가능함.
  • 지역변수 : 함수 안에 변수를 선언한 것. 함수 내에서만 사용가능한 변수

# 객체

// 빈 결과값 나온다. 왜냐하면 전역객체인 window에 window.name 있기 때문에~!
console.log(name)

// 객체
// 함수, 배열 도 다 객체이다.
let human = {
	username: 'shin inho',
	age: 32
};
// 함수의 이름이 필요없을 경우엔 생략이 가능하다.
human.introduce = function () {
	// this : 각 객체를 나타낸다. 여기서는 human을 가르킨다.
	console.log('안녕? 나는 '+this.username+'야. 나이는 '+this.age+'살이야')
};
human.introduce();

# 생성자

// 생성자(constructor)
function Person(username, age) {
	this.username = username;
	this.age = age;
	this.introduce = function () {
		console.log('안녕? 나는 ' + this.username +'이야. 나이는 ' + this.age + '이야.');
	}
}

// 인스턴스(instance)
var p1 = new Person('수진', 33);
var p2 = new Person('인호', 32);
p1.introduce();
p2.introduce();

# <script> 의 위치에 대한 고찰

  • 브라우저는 document를 차례대로 읽어 나가기 때문에 <script> 가 <head>에 있는 경우, <body> 안의 내용을 읽기전에 실행된다.
  • 그러므로, </body> 바로 위에 위치시킬 경우엔 문서를 읽고나서 <script> 가 실행 되어 위치가 좋음.
  • 문서가 뜨고나서 script가 읽어지므로, 사용자 입장에선 웹페이지가 더 빠르게 동작하는 것처럼 보이게 된다.
  • 만약에 <head> 안에 <script> 를 위치 시킬때는 아래와 같이 써줘야한다.
window.onload = function () {
			var title = document.getElementById('main-title');
			console.log(title);
		};
// or 위-아래 구조는 동일한 기능이다.
window.addEventListener('load', function() {
	var title = document.getElementById('main-title');
	console.log(title);
});
// or
window.addEventListener('DOMContentLoaded', function() {
	var title = document.getElementById('main-title');
	console.log(title);
});

// jQuery
$(document).ready(function () {
	...
});	// 이 구조는 jQuery 에서 이제 지원안하기로 결정함. 쓰는거 지양하도록!
// or
$(function () {
	...
});
  • load 와 DOMContentLoaded 의 차이
    • load 는 전체 이미지 포함 문서 데이터를 다 읽은 후에 함수가 실행된다.
    • DOMContentLoaded 는 DOM의 **태그 구조들(객체)**이 로드 된다면 바로 실행이 된다.

# 선택자

var title = document.getElementById('main-title'); // id 셀렉터
console.log(title);
var titleSpan = document.getElementsByTagName('span') // tag 셀렉터
console.log(titleSpan[0]);
var mainTitle = document.querySelector('#main-title'); // id, class, tag 등 모두 사용 가능! 이걸 추천
console.log(mainTitle);
var imgLi = document.querySelectorAll('.img-li'); // 클래스 처럼 같은 값은 가진 모든 객체에 접근할때 사용
console.log(imgLi[0]);
  • querySelector 는 가장 첫번째만 가지고옴
  • querySelectorAll 는 모두 가지고옴 - 배열로 하나씩 접근 가능

# 속성값 불러오기, 변경하기

var link = document.querySelector('a');
// href 값 불러오기
console.log(link.getAttribute('href'));
// href 값 변경하기
link.setAttribute('href', 'https://naver.com/')
console.log(link.getAttribute('href'));

# 클래스 추가, 제거

var link = document.querySelector('#main-title');

link.classList.add('special', 'foo'); // class 추가, 추가로 넣을시 , 로 구분
setTimeout(function () {				// 2초 뒤에 함수 실행
	link.classList.remove('special'); // class 삭제
}, 2000)

# 태그 삽입

var ultag = document.querySelector('ul');
var listElem = document.createElement('li'); // listElem 을 li 로 만들고,
listElem.innerHTML = '<span>일분이</span>';  // 추가 구조는 .innerHTML 로 만든다.
ultag.appendChild(listElem); // ul 안에 listElem 삽입
