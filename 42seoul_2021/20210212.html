# 7. 학습 내용에 대한 개인적인 총평

  • table 요소, form 요소는 평소에 자주 사용하는 태그가 아니다보니 코드를 정확히 짜는데 어려움이 살짝 있었다. 구글링을 통하여 순서에 맞게 코드를 구성하는 법과 label, input 요소를 사용하면서 form 형식에 맞춰서 코드를 짜는 방법에 익숙해질 수 있었다.
  • 간만에 html, css 를 하다보니 생각보다 구성을 묶는데 시간을 썼다. 직관적으로 보이던 때와 다르게 시멘틱 요소를 사용하려 하다보니 어떤 태그가 더 어울릴지 고민을 하게 되는 시간이었다. 큰 요소에는 시멘틱 태그를 많이 사용하였지만, 안쪽으로 들어갈때는 div, span 요소를 사용할 수 밖에 없었고, 이러한 부분이 괜찮은 태그인지 고민을 계속 하면서 코드를 짰다.
  • 앞으로 계속해서 잊지 않도록 web 관련 코딩도 꾸준히 연습할 수 있도록 노력해야겠다.
