# 2021-03-09(화)

# 1. 학습 날짜

  • 2021-03-09(화)

# 2. 학습 시간

  • 11:00 ~ 12:00 (집)
  • 13:00 ~ 16:00 (집)

# 3. 학습 범위 및 주제

  • norminette 변경에 따른 libft, gnl 코드 수정

# 4. 동료 학습 방법

  • 그룹스터디 (kyukim, yuchoi)

# 5. 학습 목표

  • libft, gnl norminette v3 버전으로 수정.

# 6. 상세 학습 내용

실제 코딩 시간 : 총 0시간.

  • norminette v3 설치
    • python3 버전 설치
    • pip update
      python3 -m pip install --upgrade pip setuptools
    • install norminette
      python3 -m pip install norminette
    • norminette update (현재는 최신버전이라 안해도 무방)
      python3 -m pip install --upgrade norminette
    • 경로 alias 추가(이전 norminette 로 인해 경로 꼬임)
      # 나의 mac 기준
  vi ~/.zshrc

  alias norminette="~/Library/Python/3.8/bin/norminette"
  :!wq

  source ~/.zshrc
  • 변경된 norminette 규칙
    • ASSIGN_IN_CONTROL : 조건문, 반복문 구조에서 할당 불가능
      • 새로운 함수로 만들어서 그함수를 조건문에 이용하는 식으로 변경하자
    • 두줄로 만들때 tap 위치 정해져 있음!
    • 단일 변수 선언시에도 무조건 tap으로 띄우자

# 7. 학습 내용에 대한 개인적인 총평

  • norminette 개편으로 인해서 미리 자주 사용하는 함수들을 새롭게 변경해보았다. 새로운 시작에 앞서서 미리 준비하는 자세를 가지고 빠르게 대처하는 식으로 계속 삶을 살도록 해야겠다. 특히 gnl 의 경우, 반복문 내부에서 변수가 재할당 하는 부분이 필요했는데, 이번 norminette 에서는 이러한 부분들을 못하도록 막아놔서 코드를 생각보다 많이 수정해야됐다. 이러한 부분이 과연 효율적인지는 모르겠지만, 코드를 더 심플하게 만들기 위해서 해주는 작업이라 생각하고 적응해야겠다.
