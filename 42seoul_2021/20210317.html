# IR 기법

의미있는 이미지에 대한 대체 텍스트를 제공하는 것을 말한다.

# 이미지 스프라이트 기법

다수의 이미지를 하나의 이미지에 포함시키고 좌표와 크지지정을 통해 추출하여 사용하는 방식.

이미지를 한번만 불러오면 되어 리소스 낭비가 적고, 로드가 빠르며, 이미지 용량도 줄일 수 있다.

하지만 이미지가 하나이므로, 대체 문자를 만들어 주기 위해서 IR 기법을 사용한다. (웹 접근성을 위해서)

# 요소를 화면에서 숨기는 방법

# 사용하지 말아야 될 방법

/* 스크린리더가 인식하지 못한다. */
/* 1 */ visibility: hidden;
/* 2 */ display: none;
/* 3 */ width: 0; height:0; font-size:0; line-height: 0;

# 사용할 수는 있지만 지양해야 될 방법

/* 스크린 리더가 인식하지만 레이아웃, 성능 등 이슈 발생 */
/* 1 */ opacity: 0;/* 크기를 잡고 있기때문에 단독으로 사용 불가능 */
/* 2 */ text-indent: -9999px; /* 전체 레이어의 크기가 지나치게 크게 잡힘: 성능 이슈 */
/* 3 */ z-index: -1;  /* position 속성 추가해야함: 성능 이슈 */

# 가장 좋은 방법

.blind {
	position: absolute;
	clip: rect(0 0 0 0);
	width: 1px;
	height: 1px;
	margin: -1px;
	overflow: hidden;
}

clip: rect(top right bottom left);

네개의 좌표로 지정한 직사각형 모양대로 요소를 잘라내는 속성. position 속성 값이 absolute 혹은 fixed인 요소여야하고, 만약 overflow 속성 값이 visible일 경우에는 적용이 되지 않습니다.
