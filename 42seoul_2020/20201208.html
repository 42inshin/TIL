# printf 의 format

%[flag][width].[presision]서식지정자

# 서식지정자(format specifier)

서식 지정자 출력 데이터 형태
%c 하나의 문자
%s 문자열
%p 주소값을 16진수로 출력
%d 부호 있는 10진 정수
%i 부호 있는 10진 정수 (%d와 동일)
%u 부호 없는 10진 정수
%x 부호 없는 16진 정수 (소문자 사용)
%X 부호 없는 16진 정수 (대문자 사용)
%% 퍼센트(%) 기호 출력
%f 실수 출력 (기본: 소수점 아래 6자리 출력)

# 옵션1: 플래그(flag)

flag 설명
- 왼쪽정렬(기본: 오른쪽 정렬)
0 오른쪽 정렬일 때 공백을 0으로 매꾼다.
. .n(n; 10진수) 실수일 때 소수점 이하의 자릿 수. 일부 서식 지정자에서만 사용 가능(실수, 문자열, 정수형). 나머지는 무시
* 출력할 인수 앞에서 가변인수로 폭 or 정밀도를 부여받음

# [예시]

int		num = 7;
float	num2 = 1111; // 정밀도가 있으면 폭 무시.
printf("%0*d", 3, num); //결과 007
printf("[%*.*f]\n", 6, 4, num2); //결과 [1111.0000]
printf("[%*.f]\n", 6, num2); //결과 [  1111]
printf("[%*f]\n", 6, num2); //결과 [1111.000000]
printf("[%-4d]", 42); //결과 [42  ]
printf("[%04d]", 42); //결과 [0042]
  • 여러 개의 플래그를 동시에 사용할 수 있다.
  • 0, - 은 같이 사용할 수 없다.

# 옵션2: 폭(width)

  • 출력할 값의 최소 너비를 지정한다.
  • 부호가 없는 정수로 표현
  • * 플래그로 가변인수에 width 지정 가능.
    • 숫자, * 동시 사용 불가!
  • 옵션에 . 이 없는 경우 숫자는 width를 나타냄.
  • 옵션에 . 이 있는 경우 . 앞의 수가 width (. 뒤는 정밀도)
  • 출력값 길이 > width 값 : width 옵션 무시됨.
  • 출력값 길이 < width 값 : 옵션에 따라 공백 또는 0 채움.

# 옵션3: 정밀도(precision)

  • 출력할 값의 자릿수를 설정.
  • 앞에 마침표(.) 가 꼭 있어야 함.
  • 출력값이 정수(d, i) 일 경우 정밀도 값만큼 0을 추가함. 정밀도가 출력할 값 길이보다 작을 경우 정밀도 옵션 무시.
  • 출력값이 실수(f, e, g) 일 경우 소수점 이하의 최대 자릿수를 가리키고, 자릿수에 따라 반올림 되거나 소수점 이하에 0이 추가된다. 전체 자릿수 설정은 폭 옵션을 사용함.
  • 정밀도에 음수가 할당되면 무시.

# 7. 학습 내용에 대한 개인적인 총평

  • GNL 에서 멈추다가 printf 도 준비하면서 조금더 막혀있던 코딩이 뚫리는 기분이다. printf 는 계속적으로 bug를 찾기위해서 자주 사용하던 함수이다 보니 익숙해서 조금더 어떤식으로 구상해야 될지 머리속에서 그려진다. 하지만 가변인자 라는 처음보는 서식을 학습해야 되므로 시간적으로는 오래걸릴 것으로 예상된다.
  • GNL 뿐만 아니라 빠르게 개발자가 되기 위해선 시간적인 여유가 없다. 지금의 나이에 점점 더 초조해지는 하루이다. 뛰어난 친구들이 아래에서부터 치고 올라온다고 생각이 드니 계속해서 스트레스를 받기도 한다. 그래도 있는 시간을 최대한 모아서 과제를 지속하고 취업, 창업 등의 미래를 만들어 나가도록 하자.
  • 코딩은 역시 근육과도 같다. 조금만 쉬어도 근손실이 나듯 코딩도 초보자인 나에게 쉬는 시간은 독과 같다. 운동과 코딩을 같게 생각하고 꾸준히 많은 시간을 투자하도록 하자.
