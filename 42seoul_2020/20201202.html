# javascript 학습 방법.

  1. 자바스크립트는 브라우저에서 언제든지 사용이 가능하다. 크롬의 개발자 도구를 통해 자바스크립트 언어를 사용해보자.
  • Ctrl + Shift + I 또는 Command + Option + I
  • 개발자 도구의 Console 탭에 js 언어를 사용할 수 있다.
console.log('Hello JavaScript!');
//결과
Hello JavaScript!
  • 자바스크립트는 C 언어보다 단순해서 숫자를 연산할 경우 결과값이 바로 나온다.
console.log(1 + 2 + 3 + 4);
//결과
10
  • Codesandbox (opens new window) 사이트를 이용하면 웹 상에서 javascript를 보다 쉽게 이용할 수 있다.
  1. 변수와 상수
  • 변수는 바뀔 수 있는 값, 값을 선언하고 나서 다른 값으로 바꿀 수 있다. let 을 통해 선언함.
let value = 1;
console.log(value);

value = 2;
console.log(value);
  • 똑같은 이름으로 선언하면 오류 발생.
let value = 1;
let value = 2;
  • 상수는 한번 선언하고 값이 바뀌지 않는 값. const 를 통해 선언함.
const a = 1;

a = 2; // 오류
  • 변수 선언 중 var 라는 키워드가 있지만 최근 자바스크립트에서는 사용을 지양한다.
    • var 이 let 과 다른 주요 차이점은 똑같은 이름으로 여러번 선언이 가능하다.
    • 하지만 IE9, IE10 같은 구형 브라우저에서는 let, const 를 사용할 수 없기 때문에 Babel 과 같은 도구를 사용하여 코드를 구형 브라우저에서도 돌아갈 수 있게끔 변환 작업이 필요하다.
  1. 문자열
  • 문자열을 작은 따옴표나 큰 따옴표로 감싸서 선언함.
  • 코드의 일관성을 위해서 따옴표는 둘중 하나만 써주는게 좋음.
let text = 'Hello';
let name = "홍길동";
  • 문자열 붙이기
let a = '안녕';
let b = '하세요';
console.log(a + b);
// 결과
안녕하세요
  1. 참/거짓 (Boolean)
  • 참 과 거짓으로 true, false를 써주면 된다.
let good = true;
let loading = false;
  1. null 과 undefined
  • "없음" 의 표현으로 두가지 데이터 타입이 있다.
    • null : 값이 없다. (값이 0 이다 와 혼동 주의)
    • undefined : 값이 설정되지 않은 상황.
  1. 연산자
  • 대입연산자 : = += -= *= /=
  • 산술연산자 : + - * / ++ --
  • 논리연산자 : ! && ||
    • 연산 순서 : NOT(!) -> AND(&&) -> OR(||)
  • 비교연산자 : ===, ==
    • === : 두 값이 일치하는지 확인. 타입 검사 (O) //권장
    • == : 두 값이 일치하는지 확인. 타입 검사 (X)
    • !== : 두 값이 일치하지 않는지 확인. 타입 검사 (O) //권장
    • != : 두 값이 일치하지 않는지 확인. 타입 검사 (X)
  1. 주석
  • // : 한줄 주석
  • /* */ : 여러줄 주석

# 7. 학습 내용에 대한 개인적인 총평

  • js 강의를 들으면서 기본적인 js 사용방법에 대해서 학습을 하였다. js 의 경우 C 언어에 비해 비교적 쉽게 사용할 수 있고 컴파일 단계를 필요로 하지 않아서 웹브라우저 내에서도 사용이 가능하다. 이러한 부분에 있어서 앞으로 점점 더 유용하고 사용빈도가 높은 언어라고 할 수 있을 것 같다. C언어의 경우 처음 배우는 부분이 어렵지만 자유자재로 다룰 수 있는 경우, 컴퓨터에 대한 폭넓은 이해를 학습할 수 있으므로 두가지를 병행하도록 해야겠다.
  • 코로나가 점점 심해지면서 클러스터에 점점 나가기 어려워진다. 사람들을 모아서 학습에 지쳐 늘어지지 않도록 힘내자. 지금의 견뎌 이겨낸다면 더 큰 사람으로 성장할 수 있지 않을까. 모두가 힘든 시기이지만 이 시기를 통해서 밝은 내일을 만들어 나가보자.
