# 2020-11-10(화)

# 1. 학습 날짜

  • 2020-11-10(화)

# 2. 학습 시간

  • 10:00 ~ 12:00 (집)
  • 13:00 ~ 22:00 (클러스터)

# 3. 학습 범위 및 주제

  • 부스트코스 모두를 위한 컴퓨터 과학 (cs50 2019) 영상 시청.
  • 42서울 주관 리그오브레전드 롤 대회 참가.

# 4. 동료 학습 방법

  • heom, yeonkim, gihkim, jiseo 와 5인 그룹으로 롤 대회 참가를 하여 장기간 동안의 호흡을 맞춤.
  • 9시간 가량의 시간 동안 호흡을 맞추어 전체적인 일정을 진행함.

# 5. 학습 목표

  • 자료구조에 대한 이해와 malloc, 포인터, 구조체, 연결 리스트에 대한 지식 정리.
  • 순위권 내에 들 수 있는 역량을 기르기 위해 최선의 팀 조합을 구성하여 승리하기.

# 6. 상세 학습 내용

실제 코딩 시간 : 총 30분.

# 포인터에 대한 고찰

int main(void)
{
    int *x;
    int *y;

    x = malloc(sizeof(int));

    *x = 42;
    *y = 13;
}
  • 위에 코드의 문제점은 포인터 y 를 선언하고 y 가 가르키는 곳에 13을 넣어줬지만, y 가 포인터로만 선언되었을 뿐, 어디를 가르킬지 정의되지 않은 상황이다.
  • 따라서 *초기화 되지 않은 y 는 임의로 어딘가를 가르키고 있을 수 있으며, 그 곳에 13이라는 값을 저장하는 것이 오류를 발생시킬 수도 있음!
  • 그렇기 때문에 y 가 가르킬 수 있는 곳을 만들어주는 것이 중요하다.
  • 포인터를 많이 사용했지만, 기초적인 문제에서 부터 실수를 자주하는 코드를 짜게 되므로, 이부분을 명확하게 집고 가자.

# 롤 대회 참가

  • 7경기를 진행하기 위해서 1시에 와서 리그오브레전드 게임을 goinfre 폴더에 설치를 진행함.
  • 전날 3경기를 통해 호흡을 맞춰보고 난후 2시 정도에 첫경기를 시작함.
  • 각자의 역할 속에서 최대한의 역량을 보여주어 전반적으로 준수한 성적을 낼 수 있었음.
  • 최종 스코어 4승 3패로 리그 4위를 거둘 수 있었음.

# 7. 학습 내용에 대한 개인적인 총평

  • 기존에 알고 있었지만, 계속되는 과제 속에서 기본적인 이해부분이 미흡했던 것을 느꼈다.

  • main.c 파일을 만들 때마다 컴파일 오류를 내던 부분이 많았었는데, 포인터라는 개념을 지금까지 잘못 알고 문제를 계속 풀기만 했던 것 같다.

  • 구조체와 연결리스트에서 NULL 포인터 개념을 많이 사용하며, libft 에서도 NULL, NULL 포인터의 차이를 집고 넘어가듯, 기본이지만 헷갈리기 쉬운 목록들을 정리하여 수시로 다시 보는 습관을 길러야 되겠다.

  • 리그오브레전드라는 게임을 통해서 클러스터에서 하루종일 코딩이 아닌 게임으로 시간을 보내는 것은 정말 신선한 경험이다.

  • 지금까지 클러스터라는 곳을 이용하면서 제한적인 사항이 많아 편한 느낌은 아니였는데, 온종일 게임을 하고 그곳에서 사람들과 소리를 내면서 호흡을 맞추고, 결과에 웃고 울 수 있는 이색적인 경험은 한번쯤은 모두들 느껴보면 좋은 경험일 것 같다.

  • 처음보는 4명의 카뎃 분들과 러쉬처럼 각자가 맡을 역할을 나누고 승리라는 목표를 위해서 유기적으로 움직이는 모습은 마치 현업에서 여러 사람들과 팀을 구성해서 목표를 향해 집중하여 결과를 만들어 내는 부분과 유사하게 볼 수 있다.

  • 또 다른 기대효과는 계속되는 경기를 통해서 오랫동안 클러스터에 있을 수 있다라는 경험과 장시간의 게임을 통해서 앞으로 게임보다 코딩에 더 집중할 수 있도록 되었다는 점이다.

  • 처음에는 대회라는 것에 부담이 컸지만, 우리가 42서울에서 하는 대회는 결과를 내기보다는 과정과 그를 통해 알아가는 사람들, 경험들이 더 우리를 이롭게 만들어 줄 것이라 생각한다.

  • 앞으로 계속되는 이벤트들에도 기분 좋게 참가를 하도록 마음가짐을 긍정적으로 넓히도록 해야겠다.
