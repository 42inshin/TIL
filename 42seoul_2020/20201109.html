# libft function 정리

void			*ft_memset(void *b, int c, size_t len);
  • unsigned int 로 형변환 된 c 를 len 바이트 만큼 string b에 채웁니다.
  • NULL 종료 문자를 검사하지 않습니다.
  • 반환값 : 첫번째 매개변수(argument) 즉, b 를 반환합니다.
void			ft_bzero(void *s, size_t n);
  • string b 에 0 을 n 바이트만큼 채웁니다.
  • 만약 n 이 0 이면 아무것도 하지 않습니다.
  • memset(s, 0, len) 으로 구현 가능합니다.
void			*ft_memcpy(void *dst, const void *src, size_t n);
  • src 를 가르키는 곳 부터 n 바이트 만큼 dst 가 가르키는 곳에 복사합니다.
  • NULL 종료 문자를 검사하지 않습니다.
  • dst 와 src 가 overlap 될 때의 동작은 정의 되지 않습니다.
  • n 바이트 만큼 복사를 하기 때문에 unsigned char * 의 새로운 메모리 영역을 위한 포인터를 만들어 1바이트씩 값을 복사하는 것을 기억하세요.
  • 반환값 : The original value of dst
void			*ft_memccpy(void *dst, const void *src, int c, size_t n);
  • memccpy() 함수는 src 문자열에서 dst 문자열로 바이트 를 복사합니다.
  • 반환값
    • 문자열 src에서 문자 c (unsigned char 로 변환 됨) 가 있는 경우,
      • 복사가 중지되고 문자열 dst에 c를 복사 후 그 다음 바이트에 대한 포인터(dst + i + 1)가 리턴됩니다.
    • 문자열 src에서 문자 c 가 없는경우,
      • n 바이트 만큼 복사되고, NULL 포인터를 반환합니다.
void			*ft_memmove(void *dst, const void *src, size_t len);
  • memmove() 함수는 문자열 src에서 문자열 dst로 len 바이트를 복사합니다.
  • memcpy() 와 마찬가지로 unsigned char * 형변환 해야합니다.
    • 메모리 영역에 값을 넣기 위해서 해주는 작업
  • 두 문자열이 겹칠(overlap) 수 있는 경우에도 복사는 항상 비파괴적인 방식으로 수행됩니다.
  • 그러므로 src 와 dst 의 위치에 따라 복사하는 방식이 달라집니다.
    • dst 가 src 보다 먼저 있는 경우, 순방향으로 복사
    • dst 가 src 보다 나중에 있는 경우, 순방향으로 복사하면 overlap 될 수 있으므로 역방향으로 복사
void			*ft_memchr(const void *s, int c, size_t n);
  • memchr() 함수는 문자열 s에서 c (부호없는 문자로 변환 됨)의 첫 번째 항목을 찾습니다.
  • 반환 값 : memchr() 함수는 찾은 바이트에 대한 포인터를 반환하거나 n 바이트 내에 해당 바이트가 없으면 NULL을 반환합니다.`
int				ft_memcmp(const void *s1, const void *s2, size_t n);
  • memcmp() 함수는 바이트 문자열 s1을 바이트 문자열 s2와 비교합니다.
  • 두 문자열 모두 길이가 n 바이트 인 것으로 간주됩니다.
  • memcmp () 함수는 두 문자열이 동일하면 0을 반환하고, 그렇지 않으면 처음 두 개의 다른 바이트 간의 차이를 반환합니다 (부호없는 문자 값으로 처리되어\ 200 '이\ 0'보다 큼, 예). 길이가 0 인 문자열은 항상 동일합니다. 이 동작은 C에서 필요하지 않으며 이식 가능한 코드는 반환 된 값의 부호에만 의존해야합니다.

# 7. 학습 내용에 대한 개인적인 총평

  • 한달 가까이 libft를 진행하면서 다시 예전에 만들었던 함수를 보았을 때 새롭게 느껴지는 함수들이 많았다. 그만큼 지금 코딩에 노력을 덜 쏟고 있다고 볼 수 있다. 앞으로 나아가는 방향에 목적지가 보이지 않다보니 추진력을 얻기 힘들때가 많고 무기력해지는 기분이든다. 열심히 잘하자, 이런 말들이 나를 불태워 더욱더 에너지를 고갈시키는 기분도 드는 요즘이다. 학습적으로 빈약한 상태에서 좀더 긍정적인 상황들이 일어날 수 있도록 심심치 않게 나에게 긍정적인 경험을 줄 수 있도록 나를 많이 아껴야겠다.
