# WINDOWS - LINUX 설명

  • junhykim, kyukim 님 함께 카페에서 만나 지금까지 진행했던 개인 피씨에 대한 최적화 방법을 알려드렸다.

  • LINUX를 부트캠프처럼 피씨에 새롭게 깔수도 있지만, 윈도우10 에서는 앱으로 쉽고 간편하게 LINUX 환경을 쓸 수 있도록 지원하고 있다.

  • 아래의 방법을 통해 LINUX 환경을 만들어 보자.

    # 기본 설정

    • 먼저, 윈도우에서 linux 를 사용하려면 "Windows 기능 켜기/끄기" 에서 "Linux용 Windows 하위 시스템" 을 체크해야 한다.

    windows + q -> "Windows 기능 켜기/끄기" 검색 -> "Linux용 Windows 하위 시스템" 체크 후 확인

    • 잠깐의 시간동안 컴퓨터가 설정을 완료하고 재부팅을 하라고 창이 뜬다.
    • 재부팅을 하자.

    # Ubuntu 설치하기

    • "MicroSoft Store" 앱에서 "Ubuntu" 를 검색하여 설치를 진행한다.
    • 설치를 하기 위해선 마이크로소프트 계정 로그인이 필요함.
    • LTS 버전과 일반버전이 있는데, LTS 버전은 최대 5년까지 꾸준한 기능 및 보완 업데이트를 하므로 LST 버전을 설치하는 것이 유리함.

    LTS 와 일반버전 차이 : https://jootc.com/p/201905102810

    • Ubuntu 20.04 LTS 를 설치함. (현 시점 최신 LTS)

    # 기본 Ubuntu 설정

    • 기본 설치를 하면 username 과 password를 입력하여 기본 세팅을 해야한다.
    • 또한, 기본적으로 vim, gcc 등이 설치되어 있지 않으므로 명령어를 입력하여 설치해줘야한다.
    • 방금 설치를 했기 때문에 업데이트부터 진행하자.
    • sudo apt-get update 입력.
    • 비밀번호를 입력하여 설치를 진행해 주자.
    • sudo apt-get install vim 입력.
    • 같은 방법으로 gcc 도 설치.
    • sudo apt-get install gcc 입력.

    # norminette 설정

    • 추가로 자주 사용하는 norminette 도 설치해주면 개인 pc에서도 norminette를 사용할 수 있다.

    normintte 설치 : https://github.com/hivehelsinki/norminette-client

    • 윈도우의 경우 추가적으로 설치할 것들이 있음.

    • Windows

      • Install ruby 2.7 without DevKit: https://rubyinstaller.org/downloads/
      • git clone this repository or download as a zip file
      • extract into C:\norminette
      • Go inside this folder with cmd.exe
      • Run: bundle
      • Add C:\norminette into your PATH so you can use norminette.rb from any folder. (https://docs.alfresco.com/4.2/tasks/fot-addpath.html)

    • 현재 기존 hostname 이 막혀있어 오류가 날 수 있다.
      (kyukim 님께서 이 문제로 norm 설치가 안된다고 도와달라 요청하심.)

    • 기존의 hostname을 새로운 hostname으로 바꿔주자.

    $ vi ~/.norminette/config.conf

  hostname = norminette.42network.org // 로 고쳐주자.
    • 오류가 사라지는 것을 확인할 수 있었다. 잘 작동함.

    # 추가 최적화

  • vi ~/.vimrc 에 기본적인 사항을 추가해서 편하게 작업할 수 있도록 만들어 준다.

      " syntax Highlighting

  3         syntax on
  4     endif

  7 set cindent

  12
  13 set ts=4 " Tab 너비
  14 set shiftwidth=4 "자동 인덴트 너비
  15
  16 colo molokai "테마 설정
  17
  18 set visualbell

    참고사이트 : https://medium.com/sunhyoups-story/vim-%EC%97%90%EB%94%94%ED%84%B0-%EC%9D%B4%EC%81%98%EA%B2%8C-%EC%82%AC%EC%9A%A9%ED%95%98%EA%B8%B0-5b6b8d546017

  • vi ~/.bashrc 또는 vi ~/.zshrc 설정

    • 매번 사용하는 단축키를 설정하여 편하게 작업할 수 있는 환경을 만들어 주자.
    • git 관련 단축키 생성 : ga, gc, gs, gp, gl 등등
    • export MAIL="인트라아이디@student.42seoul.kr"
    • alias norminette="~/.norminette/norminette.rb"
    • alias open="explorer.exe" // 탐색기 열기
    • alias cc="gcc -Wall -Wextra -Werror" 등등..

# 7. 학습 내용에 대한 개인적인 총평

  • 기존에 몇시간에 걸쳐서 알게된 작업들을 다른 학생분들에게 알려주면서 보람을 느꼈다.
  • 가르쳐 주면서 더 기본적인 개발자 소양에 다가가는 듯한 기분이 들었다.
  • Black-Void-Zsh 이라는 테마도 알려주어 필요하다면 설치할 수 있도록 하였다.
  • 세팅이라는 부분은 계속 만지면서 불편한 부분은 편하도록 만들어주는 것이다. 이러한 부분을 초보자들은 감수하면서 계속 진도를 나가며, 그 과정에서 불필요한 스트레스를 느끼는 경우도 있다. 이런 정보를 서로간에 정보를 공유하여 더욱더 코드에 재미를 느낄 수 있도록 했다는 점에서 뜻 깊은 시간이었다.
