# 2020-11-04(수)

# 1. 학습 날짜

  • 2020-11-04(수)

# 2. 학습 시간

  • 16:30 ~ 18:00 (카페)
  • 19:00 ~ 23:00 (클러스터)

# 3. 학습 범위 및 주제

  • 기존에 만들었던 함수들 오류 수정.
  • 워머신, libftest, unit-test 3가지 테스터를 이용.

# 4. 동료 학습 방법

  • yuchoi, junhykim 과 함께 libft 학습 및 오류 수정.
  • 연결리스트에 대해서 서로간의 생각과 정보 공유.
  • printf 평가 2회 실시.

# 5. 학습 목표

  • 연결 리스트에 대한 추가 학습 및 토론.
  • 테스터를 통한 기존 함수들 오류 수정.

# 6. 상세 학습 내용

실제 코딩 시간 : 총 3시간.

# ft_split

  • 오류 : 함수의 길이가 25줄을 넘음
    • 기존에 있는 split 코드를 따로 빼내어 함수로 구현.

# ft_strmapi

  • 오류 : malloc 이 제대로 동작하지 않음.
    • strdup 를 이용하여 구현하였으나, 현재의 tmp 에 메모리 동적할당을 하지 않은 것이 문제였음.
    • malloc 을 사용하여 다시 구현하여 오류 처리.

# ft_putendl_fd, ft_putstr_fd

  • 오류는 아니지만 널 값이 들어갔을 때, 보호되는 처리를 하였는데 다시 그 내용을 없애는 것으로 변경함.

# printf 평가

int printf ( const char * format, ... );
  • 아직 한참 남은 printf 평가가 2번 잡혀서 진행하였다.
  • printf 의 경우, %d, %s 등의 서식문자에 flag,width,length 라는 특정 값을 넣어 줄 수 있는데, 이 처리가 주요 핵심이었다. 물론 각각을 구현하는 것도 엄청난 시간과 공이 소요 되는 것을 평가를 진행하면서 확인 할 수 있었다.
    • %[플래그(flag)][폭(width)][.정밀도][크기(length)]서식 문자(specifier)

# 7. 학습 내용에 대한 개인적인 총평

  • 평가를 두번하면서 속도가 빠른 2기 분들이 벌써 printf 를 끝냈다는 사실에 조금은 충격을 받았다.
  • 너무나도 느린 현재의 나의 진도 상황을 보면서 반성을 하게 되었다.
  • 수많은 노력을 쏟아 부으면서 멀찍이 달려가는 사람들을 보면서 재능과 노력을 모두 가진 분들의 모습이 대단하기도 하다.
  • 앞으로 나가는 나를 상상하며, 지금의 상황에서 타파하여 c 언어와 알고리즘을 정복해 나가도록 해야겠다.
  • 몇개 남지 않은 bonus 파트를 마무리 짓고, 전체 점검을 한 후, 0번째 서클을 완료해보도록 하자.
