# 6. 상세 학습 내용

  • 규남님의 연락으로 git server의 지원금 지급 기준 안내에 대해서 알게되었습니다.
  • 지원금 기준 방식이 다음주에 더 보완되어 올라 올거라는 소식을 듣게 되었고,
  • Slack에 마크다운에 대해 검색해보라는 글이 생각나 마크다운 구글링 하였습니다.
  • 검색을 기반으로 기본적인 마크다운 사용법 학습을 하였습니다.
  • 직접 레포트 작성 연습을 통해서 사용법을 배웠습니다.

# 7. 학습 내용에 대한 개인적인 총평

  • Bad : 올라온 바로 직후 살펴보지 않아 놓칠뻔한 정보가 있었습니다. 다음에는 공지가 올라올 때 바로 확인하고, 실행하는 자세를 가져야겠습니다.
  • Good : 학습 후 바로 익히는 자세를 통해서 머리 속에 담아두고 다음으로 미루지 않고 바로 연습해보는 자세는 칭찬해줄만 하다.
